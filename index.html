Project 3
Bug Tracker

C#

This is just like the bug/ticket tracker programs that every business uses. There is a login page that has verification and authentication protocols for employees to log in. The registration page is for new users, a report page is to report a bug/ make a ticket, the list page shows all the reports you have made with updated status changes. The user can input comments about the report or see if the IT department has sent any questions regarding the issue. The login page separates between normal users and IT members and sends IT staff to the report list that shows all reports. When the status is changed to "Finished" the reporting user has an email sent to them that the ticket is resolved and closed. The program also has a "dark mode" and "light mode" feature. It uses C# .Net as the framework also MySQL for the multiple databases.
